== Installation ==

1. Go to the Plugins Menu in WordPress.
2. Search for **"Ace Social Chat."**
3. Click **"Install."**
4. Click **"Activate."**
5. **"Ace Social Chat"** menu added to the Dashboard Sidebar.
6. Go to the **"Ace Social Chat"** options and customize.

== Frequently Asked Questions ==

= Can I add multiple agents? =

Yes, you can add "multiple agents" on Ace Social Chat.

= Does Ace Social Chat save any personal data? =

No, Ace Social Chat doesn't save any personal data and also doesn't use cookies.

= Will my WhatsApp number be visible to all users? =

No, only when a user opens their WhatsApp for a chat with the agent, they will see the WhatsApp number of that agent. Without opening the inbox, no user can see the WhatsApp number. This is to ensure the privacy of the WhatsApp number.
